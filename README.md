# MIKO Ostrzenie Narzędzi

Repozytorium aplikacji/strony dla **MIKO Ostrzenie Narzędzi** — lokalnej usługi profesjonalnego ostrzenia narzędzi w Warszawie.

Projekt startowy jest przygotowywany w Lovable:

- Lovable project: https://lovable.dev/projects/fd3cd6de-9d28-4830-ac18-17839ac6ade2
- Repo: `KumatyTomi/miko-ostrzenie-narzedzi`
- Branch główny: `main`

## Cel projektu

Strona ma działać jako premium wizytówka i prosty system zapytań/rezerwacji dla klientów, którzy chcą oddać narzędzia do ostrzenia.

Docelowi klienci:

- fryzjerzy i barberzy,
- groomerzy,
- kucharze i restauracje,
- rzemieślnicy,
- klienci lokalni z Warszawy.

## Zakres MVP

- landing page po polsku,
- sekcja usług,
- orientacyjny cennik,
- formularz zapytania/rezerwacji,
- sekcja procesu obsługi,
- FAQ,
- dane kontaktowe,
- układ mobile-first,
- stylistyka premium: ciemne tło, czerwień, metal, iskry, motyw piły/ostrza.

## Formularz kontaktowy / rezerwacyjny

Planowane pola:

- imię i nazwisko,
- telefon,
- e-mail,
- typ narzędzia,
- liczba sztuk,
- preferowany termin,
- odbiór/dostawa,
- dodatkowe uwagi.

## Przyszły model danych

Do rozbudowy po MVP:

- `bookings` — zgłoszenia/rezerwacje,
- `services` — usługi ostrzenia,
- `pricing_items` — pozycje cennika,
- `testimonials` — opinie klientów,
- `gallery_items` — zdjęcia przed/po,
- `contact_messages` — wiadomości z formularza.
